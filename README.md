# Prefix Calculator Project

This project is a prefix calculator writen in Racket. The program reads an expresion from the user and evaluates it. The result is stored in a history so previous answers can be used again.

## Files

calculator.rkt  
Main program that handles parsing the expresion and calculating the result.

mode.rkt  
Handles switching between interative mode and batch mode depending on command line arguments.

devlog.md  
Development log where I wrote about the progress while working on the project.
